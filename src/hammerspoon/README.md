# hammerspoon

Global hotkey (`Cmd+Shift+R`) and menubar indicator for toggling OBS recording.

## Setup

1. Install [Hammerspoon](https://www.hammerspoon.org)
2. Copy `hammerspoon_init.lua` to `~/.hammerspoon/init.lua`
3. Make sure `obs_teams_record.py` is at `~/bin/obs_teams_record.py`
4. Reload Hammerspoon config (*Hammerspoon → Reload Config*)

## Hotkey

| Keys | Action |
|------|--------|
| `Cmd+Shift+R` | Toggle OBS recording (start / stop) |

Menubar shows `🔴 REC` while recording, `⚫ REC` when idle.
